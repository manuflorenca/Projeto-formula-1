# 🚀 API Fórmula 1

Uma API leve e rápida para gerenciar informações sobre equipes e pilotos de Fórmula 1, construída com **Fastify**.

## 📦 Tecnologias Utilizadas

- **Node.js**
- **Fastify**
- **CORS**

📡 Endpoints
